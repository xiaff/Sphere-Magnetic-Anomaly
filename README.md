## 一、球体、圆柱体磁异常正演
### 球体参数
- 地磁场：T=50000nT
- 磁倾角：I=60°
- 位置：(0,0,-30)
- 半径：R=10m
- 埋深：D=30m
- 磁化率：k=0.2（SI）
- 剖面磁化强度水平投影夹角：A′=0
### 圆柱体参数
 - 位置：(0,0,-30)
 - 球体半径：R=10m
 - 球体埋深：D=30m
 - 磁化率：k=0.2
 
## 二、向上延拓
### 两个水平圆柱体参数
- 有效磁化倾角：is=pi/2
- 磁化强度：M=0.7A/m
- 水平圆柱体1半径：R1=10m
- 水平圆柱体2半径：R2=30m
- 水平圆柱体1埋深：D1=20m
- 水平圆柱体2埋深：D2=50m
